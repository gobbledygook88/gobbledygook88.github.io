Setup

Building from source is very easy.

Basics

Starting up

The most basic code-base for a program written in C is shown below, with annotations:

package main

import "fmt"

func main() {
    var x string = "Hello world!"
    fmt.Println(x)
}

To compile and run the program, use the following commands in a terminal

> gofmt -w hello_world.go
> go run hello_world.go

To build and/or install an executable binary, we can use the following:

> go build -v .
> go install -v .

Comments

Every package should have a package comment. It should provide imformation to the package as a whole as it appears first on the godoc page. gofmt will fix any formatting in the comments.

Doc comments are comments preceding a top-level declaration. Every exported (capitalized) name in a program should have a doc comment.

// Single line comment
/* Multi-line comment */

The godoc command extracts and generates documentation for Go programs.

> godoc package_name
> godoc package_name method_name
> godoc -http=:6060

Variables

Golang has the usual built-in types:

  • Integers ** uint8, uint16, uint32, uint64 ** int8, int16, int32, int64 ** byte == uint8 ** rune == int32 ** uint, int, uintptr (These are machine dependent)
  • Floating point numbers ** float32 (single precision) ** float64 (double precision) ** complex64 ** complex128
  • string
  • Booleans ** bool (with true and false)

To declare a variable, we use the following syntax

var i int
var i int = 10
i := 10
var (
    a = 5
    b = 10
    c = 15
)

Variable names must start with a letter and may contain letters, numbers or the _ (underscore) symbol.

Declarations may be prefixed with the keyword const, instead of the var keyword, to denote a constant.

Conditionals

if i % 2 == 0 {
    // Do something
} else if i % 3 == 0 {
    // Do something else
} else {
    // Do another thing
}

Loops

Go only has one type of loop, but it can be used in a variety of ways.

i := 0
for i < 10 {
    // do something
    i = i + 1
}

or

for i := 0; i < 10; i++ {
    // do something
}

Switch statement

switch i {
    case 0: // do something
    case 1: // do another thing
    case 2: // do yet another thing
    default: // do the default thing
}

Arrays

Arrays in Go look like this:

var x [5]int
x[4] = 100  // [0 0 0 0 100]

We can loop over arrays with a special form of the for loop:

for i, value := range x {
    // do something
}

This will throw an error upon compiling the program since the variable i is not used. We solve this by using an underscore (_).

for _, value := range x {
    // do something
}

There is a shorter syntax for arrays:

x := [3]int{ 10, 20, 30 }

Slices

A slice is a segment of an array; its length is allowed to change. Every slice is associated with some array. They can not be longer than the array, but they can be smaller.

var x []float64  // or
x := make([]float64, 5)  // to create a slice of length 5, or
x := make([]float64, 5, 10)  // where 10 is the length of the underlying array

To create a slice from an existing array, we can use the [low : high] syntax.

arr := [5]float64{1,2,3,4,5}
x := arr[1:4]  // [2,3,4]
y := arr[0:]   // [1,2,3,4,5]
z := arr[:4]   // [1,2,3,4]

The append() function creates a nice slice by taking an existing slice (the first argument) and appending all the following arguments to it.

The copy() function copies elements from one slice to another

slice1 := []int{1,2,3}
slice2 := make()[]int, 2
copy(slice2, slice1)  // [1,2]

Maps

A map is an unordered collection of key-value pairs.

x := make(map[string]int)
x["key"] = 10
len(x)
delete(x, "key")
